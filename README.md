# Interactive AI Teaching

### Mapas de conocimiento interactivos y curados para ingeniería de IA en producción.

Esta plataforma es un recurso educativo diseñado para ingenieros que buscan profundizar en la arquitectura y operación de sistemas de inteligencia artificial modernos. A través de interfaces interactivas y visuales, desglosamos conceptos complejos desde la infraestructura hasta la optimización en producción.

---

## Módulos Actuales

- **End-to-End RAG Engineering**: Un recorrido profundo por el ciclo de vida de un pipeline RAG: desde estrategias de chunking y anisotropía hasta evaluación avanzada con RAGAS.
- **AgentOps (Agent Lifecycle)**: El mapa definitivo para diseñar, desarrollar, desplegar y optimar agentes inteligentes autónomos de grado empresarial.

---

## Stack Tecnológico

- **Core**: React 19 + TypeScript 6
- **Build System**: Vite 8
- **Styling**: Vanilla CSS (CSS Variables + BEM) — *Zero external UI libraries for maximum performance and design control.*
- **Deployment**: Vercel

---

## Despliegue

```bash
npm install
npm run dev     # Desarrollo local
npm run build   # Build de producción optimizado para Vercel
```
